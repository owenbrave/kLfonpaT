# 项目简介

## 核心功能
- 学生学习成果展示
- 开题报告及任务书管理
- 管理员、教师、学生多角色权限管理
- 前后端代码及SQL文件提供
- 万字文档详细说明
- 参考导入调试视频、讲解视频及开发工具安装包

## 适用场景
- 教育机构学生学习成果展示
- 教师教学成果记录与分享
- 学生个人学习成果展示与记录

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
